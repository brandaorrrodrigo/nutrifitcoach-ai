# 🥗 NUTRIFITCOACH

Sistema profissional de nutrição com IA, anamnese completa e análise de exames.

## 🚀 Quick Start
```bash
npm install
npx prisma generate
npx prisma db push
ollama serve
npm run dev
```

## ✨ Funcionalidades

- ✅ Anamnese profissional (10 etapas + 70 campos)
- ✅ Análise IA de exames laboratoriais
- ✅ Geração de cardápios personalizados
- ✅ Gamificação completa
- ✅ WhatsApp + Telegram bots
- ✅ Notificações + PWA
- ✅ Multi-idioma

## 📊 Diferencial

**Anamnese mais completa do mercado:**
- Upload de exames reais
- Análise automática de resultados
- Alertas de risco
- Recomendações personalizadas
- 12 marcadores laboratoriais
- Antropometria completa

## 🎯 Análises IA

- 🩸 Exames: glicemia, colesterol, tireoide, vitaminas
- 📏 Antropometria: IMC, circunferências, % gordura
- ⚠️ Riscos: cardiovascular, metabólico, nutricional
- 💡 Recomendações: baseadas em tudo acima
